# WebPack-SPA

Website template for single page application with client site routing and TypeScript, Still in progress

## Getting Started

#### Download Dependencies ####

```bash
npm install
```

#### Start Dev Server ####

```bash
npm start
```

#### Build App ####

```bash
npm run build
```
